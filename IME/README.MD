ğŸš€ Penggunaan API

Endpoint

POST /extract

Request

- Method: POST
- Content-Type: multipart/form-data
- Key: "image"
- Value: file gambar (jpg/jpeg/png)

Contoh Request (cURL)

curl -X POST -F "image=@foto.jpg" http://127.0.0.1:5000/extract

Contoh Response

{
  "analysis": {
    "is_modified": true,
    "software_detected": ["adobe photoshop"]
  },
  "format": "JPEG",
  "width": 1920,
  "height": 1080,
  "Make": "Canon",
  "Model": "EOS R5",
  "gps": {
    "latitude": -6.2088,
    "longitude": 106.8456
  }
}
